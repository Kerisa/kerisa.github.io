2019 总结

发表于 2020-01-05 更新于 2025-07-25 分类于 Life
本文字数： 859 阅读时长 ≈ 1 分钟

总结这东西应该放在年底发吧。。我也是这两天才改变主意想要写写，就变成从日记上扒点过来整合一下了。

1月份，这还算是去年的时候，那段时间过得挺压抑，因为公司收益不好眼要关门，同时也在一批批裁员，这种环境下基本没心思工作了，大家都是表面冷静地上班，自己暗自寻找出路。当然我也不例外，尽可能的在为年后的面试做准备。说实话那段时间里没什么活还是很轻松的。

年后开始上班，我也开始投简历和面试，过了半个多月差不多就敲定了，然后就是提离职报告跑路，此时公司的研发中心剩下不过十几人。

面试完，离职，但是我没有马上办入职，而是跑到同学们的城市玩了一波，之后再回到社畜生活中。之后便是毫无波澜的上班生活。

公司的出国旅行和软考时间最后撞上了，一想日本我不久前也去过这次索性就没去，选择考试了。说到为啥要考的话当初也没什么太多想法，没想靠它改变什么，可能就是…突然有兴趣了吧，有点想回味下那种考试的感觉。

夏天想到了吃灰的 PS4，把它取出来接上电源玩了几款游戏。

夏天也出去玩了一趟，在国内，多出门逛逛换换心情还是好的，就是夏天晒死个人。

还是夏天，接了天理来家里生活，从此我也是有猫的人了，哼哼。

期间梯子也不稳定，断断续续花在升级和实验上的时间也有不少了，直到最后 v2ray 总算是稳定下来，能在 4g 网络上跑了（应该不是刚好碰上解禁吧）真是浪费生命。

顺便家里也安排上了公网 ip，节约生命第二弹，Nas 得以火力全开。

秋天在看书，，，还是不这么说了吧，不然书单列出来太没存在感了（漫画算嘛？），毕竟才寥寥数本，不过花在看新番上的时间倒是挺多的。不过啃下了生肉 gal，胃口似乎变好了，原先不少等不到汉化的也可以考虑自己去推。啊不过虽然我嘴上这么说，可能也只是偶尔才会推几作吧。

11 月下旬开始继续写全明星，总算赶在元旦那几天弄完，仿佛体会到了拖稿的感觉，当然催我的是我自己就是了。

一年下来，Todo list 上的老顽固几乎没少，还新增了不少成员，面对越来越长的列表常常觉得力不从心。 干脆全删了吧从头再来（滑稽）。

这大概就是 2019 的经过了吧，充斥着懒散的单调。
