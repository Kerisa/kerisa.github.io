9-nine ゆきいろゆきはなゆきのあと 感想

发表于 2020-06-06 更新于 2025-07-22 分类于 acg ， galgame
本文字数： 5.9k 阅读时长 ≈ 5 分钟

趁着第四作发布，于是我就四作一起推了，不用蛋疼得等上一年还是挺舒服。

九次九日九重色 / ここのつここのかここのいろ

主人公新海翔所在的小镇因为一场地震导致一部分人拥有了超能力。作为被选中的人类，主人公和他的魔法少女后宫团开始与同样获得超能力的邪恶势力作斗争。超能力依附在名为 artifact 的道具上，持有 artifact 的人就可以激发它的能力使用异能。 第一作的轮班女主角是男主的同班同学九条都。九条的能力可以夺取物品的所有权（也可以是视力、记忆等没有实体的东西）。虽然男主一开始坚持自我催眠超能力是不存在的，但因为镇上出现被石化的无辜市民，男主才不得不接受。当然这时候石化事件的犯人还没有找到。很快男主又和九条一起解决了学校里少年 A 因能力失控而引发的火灾事件，最终九条夺取了他的 artifact 后平息了火灾。

不过事实上因为 artifact 被夺走，少年 A 被自己的能力反噬而完蛋了。男主回到家后，那只好几天前就赖在男主家里丢也丢不掉的玩偶终于开始说话了，她自称索菲缇娅，告诉了男主和九条关于超能力的来龙去脉，并让他俩协助回收 artifact。 在这期间男主的妹妹新海天也出现了能力，她的能力是消除存在感。还有一个学姐香坂春风，她的能力类似于魅惑，但是第一作里和男主并没有过多交流。 后来第二次石化事件发生，男主准备拉最后一位女主结城希亚入伙时，九条读取了男主的记忆时意外知道了少年 A 间接被自己杀死的事实。怀着极大的负罪感和责任感的九条开始独自搜索犯人，犯人虽然成功被九条找到，但她却独自迎敌然后被反杀。

读档。一周目的 BE 是强制进入的，二周目开始可以回避掉，关键就在于男主之前和索菲交流时不能问她夺走 artifact 后对方会怎么样，这样就算后面被九条读了记忆也不会导致九条只身涉险，最多就是暴露男主对着九条脑补过的过激画面而已。顺便一提，男主自己当然还是一直保持着无能力的状态，这就能方便后面作弊了。比如一周目这么明显的走一通 BE 再从头开始，玩家肯定能联想到男主的能力是回溯时间方面的了，有点 metagame 的味道。

通过其他途径收集到犯人情报后，男主一行便拉上希亚前往神社确认。不料犯人没有现身，神社只有一尊被石化的男子，还附着一封怎么看都很可疑的自白遗书。

虽然很想吐槽这种东西都能信，不过无法再入手新线索的男主一行只能作罢。而后明面上也没有再出现被害事件。索菲无奈表示你们太菜了这样下去别想找到其他 artifact，我就不跟你们玩了，说完后跑路了。 最后一幕冒出一个在天台上一番冷笑的金发女人，然后第一作也就到此结束了。

emmm……感觉完全是骗钱的节奏，什么也没展开，什么也没说清楚。分卷拆得这么细，剧情居然还没多少，看来是整个系列都要盯死石化事件的节奏了，我觉得大可以多安排几个事件，然后在第一作里解决一些无关紧要的事件，顺藤摸瓜摸到大 boss 不也挺好的，这样第一作看上去还能结束地圆满些。

天色天歌天籁音 / そらいろそらうたそらのおと

第二作开始把时间线又还原回去了。 这一作男主和妹妹的关系更加亲近，自然是准备发展妹妹了。 妹妹还是和第一作一样继续在动画同好网站上收集到了学姐和其他能力者有线下聚会的情报。调查在顺利进行。不过最后天尾随学姐三人组在神社的会面时被抓包了。

男主和九条赶往神社出面和三人组进行了一番友好的对话然后准备干架时，一路尾随男主们的希亚出来压住了场面。鉴于希亚和对方的队长高峰莲夜都是中二，两个对上电波后聊得还挺尽兴……三人组里除了学姐，最后一个就是第一卷末出场的幽灵（ghost），不同于中二的高峰和学姐，ghost 很具有攻击性，还不停挑衅男主。 隔了几天，男主独自跑到公园瞎晃，成功遭遇了 ghost，她在开打前还卖了一波友善。ghost 拥有多个能力，其中之一就是石化。正当男主被石化要完蛋时，索菲前来救场，但是没成功。最后躲在一旁的妹妹情急之下使用能力将 ghost 击退，也就是将 ghost 的存在（接近）消除了。

刻印蔓延后的天样子逐渐变得古怪。这个发展自然就是妹妹的能力开始不断失控，她自己的存在也在不断消失。索菲拉住男主说我这有解除魔法少女契约的药，但是还在制作中所以要等等，在这之前希望你能先稳住妹妹的情绪拖住时间。扛了几天后妹妹说，老哥我马上就要消失了，我心里有个夙愿就是想和哥哥来一发。。。 作为哥哥听到这种话还能说什么呢，那当然是义正辞严地拒绝了，说以魔朵哟，难道一发你就满足了吗，等你喝完药我们每天干个痛快好不好？（好）

等到天喝完药，ghost 也适时恢复了。男主就想装 b 趁着妹妹睡觉时把问题解决了，于是就跑去对 ghost 下手，出发前还收下了从索菲那里拿到的火焰 artifact。战斗自然打得很辛苦，后来在中途加入的学姐的帮助下，男主抓住机会果断将 ghost 烧成灰（没有劝 ghost 从良等白痴剧情，赞一下）。后面高峰赶来想补刀男主，旋即被索菲带来的后宫三人摆平。

高峰也能使用魔眼被解释成眷属化，高峰成了 ghost 的眷属后复制到了她的能力。 魔眼并没有被回收，所以真正的犯人还在。同时也直接揭露了幕后黑手深泽与一，他其实就是男主的基友，而且和索菲似乎也认识。

这一作比起第一作好上不少，如果不是非要这样搞成一位女主出一作的话把一、二卷合起来作为第一卷就挺合适。日常部分基本就是和妹妹一天到晚说相声斗嘴，日常部分就得是这种样子才行啊，完全沉迷了……

春色春恋春熙风 / はるいろはるこいはるのかぜ

游戏的套路基本都熟悉了。时间回到在神社与三人组对峙后的第二天，4.26 号。 为了和学姐构筑良好的关系，男主代替妹妹联系了学姐，并准备潜入学姐的集团。因为每次回溯仍然保有记忆，男主知道 ghost 就是魔眼使用者。成功潜入的同时，开始扮情侣假戏真做，更何况学姐的第二人格一直就想诱骗男主上床。学姐的能力是心想事成，简直就是最牛逼的场控，buff 和 debuff 随意安排，之前的魅惑只是一个学姐想引诱男主的表现效果而已。另一个人格也是学姐对自己施加能力（催眠）后的结果。 第三作的时候当然已经成功拉学姐入伙了，开会讨论时大家突然发现 ghost 长得和与一很像，并从索菲处了解到幻体的存在。与一可能是因为掌握不好幻体的使用，所以才做了个和自己很像的幻体出来，自然也就推得与一是黑幕了。 男主通过让索菲变成娃娃机里的玩偶来确认与一是否为能力者，因为只有能力者才能看见索菲。 扒出与一真身后，敌我八人都登场了，与一和莲夜完虐主角团五人一番后扬长而去。

新角色伊莉丝登场，她附身在沙月身上找到男主和学姐，因为据说成濑家代代都是伊莉丝的同一存在（指两个世界中灵魂相同的人），所以伊莉丝才能借此附身（同调）。然后伊莉丝告诉男主索菲就是想让你们相互厮杀，借此收集 artifact 用来满足私欲，提醒两人提防索菲。伊莉丝还顺便介绍了世界之眼这个 artifact，它可以连通不同的世界，也能作为“门”的存在。两个世界都各存在一个世界之眼。而因为游戏最开始的那次地震导致神器损坏，导致一直被关闭的大门洞开。另外世界之眼同样也有观测世界分支的能力，索菲就是依靠它进行观测的。说完伊莉丝便给了男主一支安布罗西亚，也就是妹妹以前用的解除契约的药，说你只要喂给基友就能不战而胜了。

男主还将信将疑，回去后问了索菲，索菲也说真有这药，但是现在没货，已经在锐意制作中了。男主和妹子们一合计就打算赶紧下药去了。 成功让与一和高峰喝下饮料后，他俩却直接凉了。伊莉丝出现悠悠地捡走与一的魔眼，然后瞬间就石化了四个妹子。显然伊莉丝才是大反派。

男主觉醒能力穿越回 5.2 号和索菲谈到关于沙月的时候。当然，此时的男主已经带上了刚刚发生的全灭结局的记忆，于是大家准备和伊莉丝干架。打斗中途与一也过来帮了忙，还把幻体和魔眼都送给了男主。男主利用 SL 大法不断回避死亡，但依然没法打出制胜一击。然后学姐在希亚的鼓动下也觉醒了，给大家上完 buff 后妹子们合力破坏了世界之眼，将伊莉丝击退。

后面还有点剧情，学姐遇上了小学时欺负她的男同学，折腾几次后又一次在公园碰上这个处于狂暴状态的龙套，然后在缠斗中被乱入的与一解决。这个龙套是伊莉丝的实验体，他虽然没有 artifact，却也能够使用能力。伊莉丝和与一实际仍处在半合作状态中。 尾声中索菲登场，她表示伊莉丝是当初自己是否战胜 artifact 的诱惑而做了不同选择后产生的平行世界里的自己。她俩相互通过世界之眼观测到了对方，但是无法干涉。同时告知玩家男主在地震中获得的能力是世界之眼的碎片，真正展现出 SL 大法（学名 Overlord）的能力者其实是电脑前的玩家。玩家和男主其实是同一存在，所以玩家可以通过男主打开的世界之门完成同调，然后干预男主的行动，是隐藏的第⑨位能力者。

这作的感觉就挺好的，不但把魔眼问题解决了，还反转出一个最终 boss，战斗描写也不错。接下来终于到收尾的最终作了。

雪色雪花雪余痕 / ゆきいろゆきはなゆきのあと

有了第三作的铺垫，第四作开始就搞了个所有分支的总览，帮助玩家回忆前作，理清脉络。

通过复盘前面的战斗，索菲发现用希亚的能力 The Order 可以真正干掉身在异世界的伊莉丝。与在索菲的建议下，玩家使用 Overlord 在最开始的 4.17 日取得了索菲的信任，使得学姐线出现了制胜的机会。Boss 战很快展开，但是在准备绝杀时希亚出了点岔子，没能出招干倒伊莉丝。那是因为希亚一直以来都和其他人若即若离（没有被攻略）。于是为了搞清希亚背后的情感，玩家在各个分支里跳来跳去，尝试对分支进行正面的干预，但是也没能分裂出成功的结果，世界线还是收束了。因此需要开辟个希亚线，搞好关系以便让希亚发挥真正实力。

希亚的能力 The Order 同样也是个 bug 存在，可以凭能力者的主观概念给对方定罪，小到定身大到抹杀，完全可以肆意妄为。 男主在这条线上尽力提前进行干预，避免出现牺牲者，包括阻止与一和救下火灾的失控者。 平静了一段时间后伊莉丝终于出现，通过将普通人 artifact 化引发了大规模骚乱，不过人们只是像丧尸一样受到摆布，并没有异能。总之在后宫团的努力下由希亚使出最后一击将附身在老师身上的伊莉丝打倒。 于是就这样迎来了结局，世界恢复了和平。

我一直在等反转，等啊等，最后 CG 都出来了，ED 都开始放了，我有点着急，并准备开始失望。没想到播到一半的 ED 开始倒带了（不知为啥看到这我心里一惊，虽然实在说不上有啥恐惧的要素，也许是 ED 画面很昏暗的缘故，再就是音效的加成。我记得巨人第三季最后一集的 ED 也玩了这个花样）。然后时间回退，事实化为乌有，一切回到了希亚绝杀伊莉丝之前（这就是被 Overlord 的感觉~）。 希亚干掉了附身与老师身上的伊莉丝，但那个只不过是个替身。随后被真身附身的与一上场了。也就是说未来的伊莉丝研究出了 Overlord，然后通过和现在的与一同调，并把她的记忆转给了与一。

然后男主就发现了与一也能用 Overlord 读档，即便男主能读档，他也没有能打倒与一的攻击手段，无能为力的他只能眼睁睁看着妹子们被一个个杀死。

接下来男主回溯时间做了许许多多的尝试，但是都没能逃脱与一的追杀。回溯到最初的那天，与一可以方便地杀死落单的妹子们。回溯到大家都聚在一起的时间，与一也照样跑来屠杀，战力的差距相差太大了。 男主改变主意想杀死获得伊莉丝记忆前的与一，计划同样也落了空，一天下来没找到一个妹子的男主心灰意冷回到公寓，迎接他的是希亚冰冷的尸体，与一还将除希亚外的 artifact 带来送给男主。 最终打破僵局是在男主搞清自己的能力之后。

男主生成了四位女主的幻体，并用世界之眼将玩家和四位女主连结，由玩家导入记忆，然后将四位女主“召唤”到了男主所在的分支。可以参照索菲操作幻体的行为，实际上是她们在各自的分支里远程操作幻体（恋爱故事的情节居然通过这种设定利用起来，我™直呼牛皮）。当然男主还有个隐藏能力，就是能完美驾驭拥有的 artifact。各个分支的妹子虽然是完全体，但是毕竟在当前世界里的只是个幻体，仍然没法解决战力短缺的问题。因为之前与一贴心地送来了妹子们的 artifact，男主一口闷了之后就开始各种输出了。还能做到让同调的学姐强化使用着学姐能力的男主这样的套娃操作。

然后男主就在天台上一次次杀死与一，吃亏的与一和伊莉丝加强同调提升能力，但是男主也及时收到了索菲送来的一票强力 artifact 继续干与一。 与一继续要求伊莉丝强化自己，于是现在的伊莉丝占据了与一的身体。未来的伊莉丝仍然藏在台后通过假魂连接与一，但是在使用 Overlord 的瞬间，未来的伊莉丝会和与一（因为强化与一的关系现在变成和伊莉丝自己了）真正连接在一起。另外想要彻底打倒伊莉丝，还需要消除所有与未来连接过的分支，这样才能消除创造出第二个 Overlord 的可能性。

伊莉丝不傻，她也想用 The Order 干掉和男主同调的玩家。然后男主及时切断了和玩家的连接。索菲便引导玩家和拥有世界之眼的沙月同调。接着玩家和沙月一起找出了过去、现在和未来上所有分支的伊莉丝，并完成了和她们的同调。条件都集齐了，最后就等 The Order 炸裂，就能一口气消灭所有的伊莉丝了。

其实我想放这张图。

干掉伊莉丝后，妹子们也纷纷下线，玩家在男主的拜托下，将胜利的记忆植回了其他分支的男主身上。 这次真的能够好好享受一切结束后的平静生活了。

在尾声中索菲说还有最后一个任务，应该是去收下后宫把最初被石化的九条救下来。根据索菲的说法推测，天和春风的 BE 都是由男主的行动分裂出来的，通过 Overlord 剪支就能够回避掉；而九条则是她自己的行动产生出的分支，这样没法被剪支。总觉得有点牵强，没被剪支影响的话后面又要怎么回避它呢，总之还是 fd 见吧。

CV 方面新井里美那独特的发音技巧辨识度太高了……意外的是男主也一下就和当妈那慵懒的声音重合在一起，一查果然是阿部敦。要是再叫上炮姐齐聚一堂就厉害了。第一作澤田的声音感觉很通透，但是第四作里听着常常会觉得沙哑，好像有点不在状态，没有前几作的那种清脆感了。

第三作和第四作的质量还是很不错的。特别像是把反转时机塞在 ED 里，准确地说是 ED 播放到沙月的人物介绍处开始倒带，时机选的很让人意外。在 ED 之前真的是一脸困惑地边猜什么时候反转边推着结局。megagame 的安排虽然较常见吧但也要比什么失忆多人格要好多了。但不得不说的是，一年一卷实在是有点过分，而且每卷的剧本也不多，前两作节奏也慢，养肥了一口气推算是万幸，不然等得多蛋疼……也许半年一卷还能等等。难道 palette 预算过于短缺，只能卖一作回血接着做续作？要这么说没被腰斩真是好事。
