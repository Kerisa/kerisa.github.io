樱之杜†净梦者——一圆三吃

发表于 2021-05-12 更新于 2025-07-22 分类于 acg ， galgame
本文字数： 2.4k 阅读时长 ≈ 2 分钟

之前没玩过老吴的作品，玩了才知道老吴爱整鬼片。上来先发了一整个序章的糖。正当我想着圆香这么牛皮，其他女主要怎么和她竞争的时候，小圆直接领便当走了……辣手摧花毫不犹豫。OP 放完我还没缓过来时，小圆已经变成慎司脑内专属灵体女友了。

不过这样安排剧情也还可以，能让小圆继续有戏份，也能保持在慎司心中的特殊地位不动摇。后面再安排起其他妹子的个人线也容易展开，虽然樱之杜的个人线只是强行凑数……

圆香变成幽灵后正篇正式开始。男主在小时飞机失事时曾看到过恶灵，而如今恶灵又在镇上出现，并引发了各种事件。因为能看到死者幽灵和恶灵，男主在同班妹子暮羽的邀请下加入樱之杜†净梦者，和其他两个妹子一起解决事件。同时男主也在追踪杀死圆香的凶手——Joker 的线索。每个事件都有不少猎奇元素，最初我也是因为看到 CG 勾起了兴趣……

首先是同班同学远野香奈突然失踪。男主看到了她的幽灵，调查发现她常去的一家餐厅的厨师有重大嫌疑。男主于是和暮羽在学姐真幌的能力下潜入梦境，和梦境里的厨师战斗，成功清除了蚕食他灵魂的红色恶灵。恶灵分两种，蓝恶灵大量出现的地方将会发生惨案，附身到人身上则会致人死亡。但蓝恶灵只能算是一种“现象”，它会被惨案吸引，但惨案并不是由它制造。而红恶灵附身到人身上会使人疯狂，并且被附身的人杀了其他人后不会留下作案痕迹，尸体会直接变成蓝恶灵。净梦者的主要目标是红色幽灵。

接着女主角之一美冬要独自调查恶灵，慎司陪她到某个废弃小屋发现了 Joker 的线索，而且又在被 Joker 发现时运气好虎口脱险。慎司因为救下了本该交待在那的美冬，使美冬的蓝恶灵附身到了自己身上。然后慎司用暮羽爷爷教的方法玩俄罗斯转盘捡回一命，然后美冬也顺势回归净梦者。

后面还有两个事件，分别是阴沉同班同学葛木和强欲萝莉学妹亚里砂。两人都吃了红恶灵一波附身。最后葛木未能按原计划用炸弹炸死全班同学（等同于救下了他们）反被蓝恶灵附身而死。 亚里砂事件里妹妹初音意外展现出特殊能力，她驱逐了附身到慎司身上的蓝恶灵。蓝恶灵旋即进入亚里砂体内，然后亚里砂照慎司说的方法，却赌命失败而死。

接着就是经常发出和 Joker 类似怪笑声的可疑恶心老师。然而老师只是个普通的男同……他倒也可以说是 Joker 的受害者。他和 Joker 出自同一孤儿院，在和 Joker 的关系中充当 1，虽然实际上克苏鲁 Joker 才是上位者……Joker 的身份逐渐浮出水面，他正是慎司身边（看上去）定位模糊的迷之朋友，整天无所事事的八束秀典。秀典经常常挂在嘴边的话就是让慎司成为英雄，后半句没说的是让英雄来惩戒自己。所以为了让慎司能这么做，他才动手杀了圆香。阐明下观众老爷们最关心的事，因为秀典拜他妈所赐变成了不举，所以圆香是清白的……

触手 Joker 显然不是慎司能单挑的，所以在亚里砂事件中安排了暮羽姐姐芹那出场，之后 Boss 本芹那挑下大梁和 Joker 刚正面，然后慎司躲在后方猥琐。

终于将 Joker 收拾了之后，慎司数年过来的痛苦和悔恨终于得到了填补，圆香大仇已报，也准备升天了。酝酿已久的感情终于在此情此景奔涌而出，圆香希望慎司能继续向前看，重新面对生活。

剩下的就是妹子倒贴的个人线了。前面也说过个人线聊胜于无，本来这种悬疑类的剧本走向就比较固定，弄些分支出来搞搞小暧昧还行，发挥太自由了就会破坏主线的统一。个人线的日常也非常单一，就是东逛西逛做饭打炮……暮羽线倒可以算是主线的延伸，交待了一些背景，还把 Joker 重新拉出来打了一顿。

第二部的时间线回到打倒 Joker，圆香升天两个月后。慎司之后并没有和四个妹子有什么进展，依然郁郁寡欢，经常对着通讯录里圆香的手机号发呆，犹犹豫豫不舍得删号码。

随后新的异变降临。净梦者众发现自己入睡后会在新的红色噩梦中醒来自由活动，也能相互看到其他路人，与现实无异。和普通人不一样，他们回到现实后仍然保持红色梦境中的记忆。 和以前的个人梦境不同，在红梦中死亡会使人在现实中做出自杀行为。此处基友盛情出演被害者。 于是净梦者开始处理搞事的人。中途还有新人物学弟春川和学长神定加入。

另一边是圆香没有遭遇 Joker 毒手的 if 线。圆香当年并没有遇害，她在搬到东京生活两年后又重新回到了樱之杜，和慎司开始恩爱的同居生活，持续不断地发糖。

两条线交替进行，成了鲜明的对比。短暂靠近的两个世界终将走向不同的分支。慎司无时无刻不在假想着圆香本应该得到的平稳的幸福，而这对他说早已是无法触及的存在。就像每次慎司梦到圆香一样，圆香在并行的电车上低头看着手机，驶向新的轨道，慎司的车则一头扎进死胡同里，两人甚至连目光都无法交汇。

两条线中慎司（if 线中和圆香一起）都在追逐“失踪 7 年的杀人魔兵藤”。if 线的展开也比较惊险，圆香和慎司两人从小妹妹摩耶的幽灵交流后知道了兵藤藏身在废弃地铁站。于是他们便摸黑下到地底去收集兵藤的证据，很快就要走向一圆四吃的局面。然而最终拯救了 if 线圆香的是另一世界的慎司在红色梦境中意外拨通的电话。

现实中的慎司已经知道了红色梦境中出现的兵藤并不是本体，也知道了梦中会错乱时空的特点。鬼使神差地，他拨打了圆香的手机号。这通搭载了无数思念的电话不辱使命，将奇迹的声音传到另一个世界，成功救下了 if 线的小圆。

（稍微补充下，引发红色梦境的真凶是当年一个因兵藤而遇害的小孩的灵体，他附身到了学弟春川身上，并且引发了红色梦境。他自己在梦境中也有很强的能力，可以消除慎司等人的记忆，所以慎司他们一直找不出这个崽种的踪迹。然后 if 线的兵藤也想干掉这个崽种，恰好红色梦境能扭曲时空，兵藤就这样从 if 线乱入过来了。）

此时的慎司也终于能够放下对圆香的遗憾，将这段最美好的回忆珍藏到内心深处，同时也有余裕来迎接新的未来。

……然后他转身就开后宫去了，前面感情酝酿了这么久到这有点脱力，一句“那就平等地爱你们四个人”我一下没反应过来，当真不挑食。哦我懂了，“反正不是圆香，换谁都一样”。
