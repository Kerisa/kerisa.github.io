Pixel 7 到了

发表于 2022-12-17 更新于 2025-07-25 分类于 Life
本文字数： 972 阅读时长 ≈ 1 分钟

之前纠结了一段时间，最终还是决定入手 pixel 尝尝鲜。随后赶着黑五下单了 pixel 7。其实在这之前我就一直用小米刷着 Pixel Experience 做主力机在用，所以 Android 13 倒没什么新鲜感，主要还是想体验 google 的 ai 相机。 另外正式迁移数据前还要先 root 处理下 VoLTE。

免 Root 开启 VoLTE

在自动升级了一次系统后发现丢 root 了，一开始试图重新 patch init_boot.img，同时又想解决不能安装 32 位 apk 的麻烦，于是尝试了这个打过补丁后的 Magisk ，结果就是 “Your device is corrupt”……随后又找到了这个免 root 的开启方法，里面介绍说是利用了 Android 的 telephony.ICarrierConfigLoader.overrideConfig() 内部 API 实现了运营商配置的改写，按他的方式操作了一遍，顺利免 root 开启了 VoLTE，Magisk 也不用再安装了。

Root

pixel 7 的 root 可以参考这篇知乎文章，步骤写得挺详细。

如果重启进入 fastboot 模式后发现 fastboot flashing unlock 命令提示 waiting for any device 说明电脑需要安装 google 的 USB 驱动，在Android 开发者网站下载驱动，再到系统的设备管理器里找到手机，手动更新驱动程序后即可继续解锁。解锁后手机 fastboot 界面下方就会显示红色的 unlocked 字样。

之后刷入经过 Magisk patch 的 boot 镜像即可完成 root。

解锁 VoLTE

重启手机后完成最基本的设置进入主界面。先把 Magisk 安装回去，然后照着文章上面第五节

Pixel 7系列解锁国内5G

的步骤写入 VoLTE 配置，修改信号制式后即可。

重启后关闭 wifi 打个电话，就能发现可以同时上网了。

最后终于可以开始迁移手机数据了。

PS：不少应用在 pixel 上不能安装，因为 Pixel 7 系列已经不再支持 32 位应用，所以对那些安装不了的 app 我们只能联系相应的客服求更新（上面也提到了非官方的方法不过没能成功，但可能只是我的操作问题，有空再折腾好了）。 PPS：新手机登录微信后为了备份消息我又切回旧手机登录了一次，然后微信就告诉我账号有风险，并且已经贴心地替我禁用三天新设备登录，是不是因为 root 了才给的特殊待遇啊(￣_,￣ )
